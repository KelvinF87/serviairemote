IA3 API Documentation

API Endpoints

POST /api/chat

Description: Sends a chat message to the AI and receives a response.

Method: POST

Authentication: Yes

Request Body:


                    {
                        "messages": [
                            {
                                "roles": "user",
                                "content": "Your message here"
                            }
                        ],
                        "modelId": "model-id-from-db"  // Optional: Specify the AI model to use
                    }

Response Example:


                    {
                        "choices": [
                            {
                                "message": {
                                    "content": "AI response here"
                                }
                            }
                        ]
                    }

POST /api/auth/signup

Description: Registers a new user.

Method: POST

Authentication: No

Request Body:


                    {
                        "name": "Your Name",
                        "email": "your.email@example.com",
                        "password": "YourPassword123",
                        "roles": "user",
                        "prompt": "DefaultPrompt",
                        "active": true,
                        "img": "URL to your image"
                    }

Response Example:


                    {
                        "_id": "...",
                        "name": "Your Name",
                        "email": "your.email@example.com",
                        "roles": "user",
                        "__v": 0
                    }

POST /api/auth/login

Description: Authenticates an existing user and returns a JWT.

Method: POST

Authentication: No

Request Body:


                    {
                        "email": "your.email@example.com",
                        "password": "YourPassword123"
                    }

Response Example:


                    {
                        "authToken": "Your JWT Token"
                    }

GET /api/auth/verify

Description: Verifies the JWT and returns the user's payload.

Method: GET

Authentication: Yes (JWT required in Authorization header)

Request Headers:


                    Authorization: Bearer Your JWT Token

Response Example:

                        "_id": "...",
                        "name": "Your Name",
                        "email": "your.email@example.com",
